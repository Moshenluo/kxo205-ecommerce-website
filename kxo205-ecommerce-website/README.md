## 📖 项目简介 / Project Overview

**Starry Sky Expeditions** 是一个专业的望远镜和光学产品电子商务网站。

**Starry Sky Expeditions** is a professional e-commerce website for telescopes and optical products.

### ✨ 主要功能 / Key Features

- 🏠 **首页** - 带有视频背景和产品展示
- 🏠 **Home Page** - With video background and product showcase

- 🛍️ **产品目录** - 6 款望远镜产品详情
- 🛍️ **Product Catalog** - 6 telescope products with details

- 🔐 **用户认证** - 登录和注册功能
- 🔐 **User Authentication** - Login and registration

- 🔍 **搜索功能** - 产品搜索
- 🔍 **Search Function** - Product search

- 🛒 **购物车** - 购物车功能
- 🛒 **Shopping Cart** - Cart functionality

## 🚀 快速开始 / Quick Start

### 方法 1：直接打开 / Method 1: Open Directly

```bash
# 克隆仓库 / Clone repository
git clone https://github.com/Moshenluo/kxo205-ecommerce-website.git

# 进入项目目录 / Enter project directory
cd kxo205-ecommerce-website

# 直接在浏览器中打开 index.html
# Open index.html directly in browser
```

### 方法 2：使用本地服务器 / Method 2: Use Local Server

```bash
# 使用 Python 启动本地服务器
# Start local server with Python
python3 -m http.server 8000

# 访问 http://localhost:8000
# Visit http://localhost:8000
```

## 📝 使用说明 / Usage Instructions

### 浏览产品 / Browse Products

1. 打开首页 / Open home page
2. 点击"Product"导航 / Click "Product" navigation
3. 查看产品详情 / View product details

### 添加到购物车 / Add to Cart

1. 在产品页面点击产品 / Click product on product page
2. 点击"Add to Cart" / Click "Add to Cart"
3. 查看购物车右上角 / Check cart icon in top right

### 用户注册 / User Registration

1. 点击"Login in" / Click "Login in"
2. 选择"Register" / Select "Register"
3. 填写信息并提交 / Fill in information and submit

## 🔧 自定义 / Customization

### 修改产品信息 / Modify Product Information

编辑产品 HTML 文件（product1-6.html）：
Edit product HTML files (product1-6.html):

```html
<h3 class="product-name">产品名称 / Product Name</h3>
<p class="product-price">$价格 / Price</p>
```

### 修改样式 / Modify Styles

编辑 CSS 文件（index.css, product.css 等）：
Edit CSS files (index.css, product.css, etc.):

```css
.product-name {
    font-size: 24px;
    color: #333;
}
```
